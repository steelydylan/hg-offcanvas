hg-offcanvas
=======
jQuery and CSS plugin to build your offcanvas website.

dependency
------
- jQuery


Usage
-------

```html
<div id="offcanvas">
	<!-- Place whatever HTML you want in here -->
</div>
<button id="offcanvas-btn-right">Open offcanvas</button>
$("#offcanvas").offcanvas({
	btn:"#offcanvas-btn-right",
	fixedHeader:"#header",
	direction:"right"
});
```
